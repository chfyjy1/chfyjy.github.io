Detailed Description

The QWaylandQuickShellSurfaceItem class provides a Qt Quick item that represents a QWaylandShellSurface.

This class is used to render wl_shell or xdg_shell surfaces as part of a Qt Quick scene. It handles moving and resizing triggered by clicking on the window decorations.

See also QWaylandQuickItem.
